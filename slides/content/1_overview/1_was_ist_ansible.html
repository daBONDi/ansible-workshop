Anforderungen

Controller

  • Linux/Unix Based OS
  • Windows (WSL) unsupported, aber funktioniert
  • CygWin unsupported, noch nicht getestet
  • Python > 2.7 / Python >3.5

Managed Linux Host

  • ssh + sftp(kann auf scp gewechselt werden)
  • Python > 2.7 / Python > 3.5
  • Zusätzliche Python Module je nach Ansible Modul

Managed Windows Host

  • WinRM Konfiguration
  • Powershell > 3.0 / .Net Framework 4.0
  • Zusätzliche Powershell Module je nach Ansible Modul
  • > Windows Server 2008
