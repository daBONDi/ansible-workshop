Wie funktioniert Ansible?

- Die Einzelkomponenten wie Ansible zusammenspielt erklären

Plays/Playbooks

- Playbook = Yamlfile - Beschreibt was auszuführen ist

Modul

- Module meist in Python(Linux),Windows(Powershell) geschrieben - Grundsätzlich kann es jede Ausführare Datei sein, die ein JSON retour gibt über den stdout

Plugins

- Bauen die Verbindung zu den Hostsystemen auf - Bestimmen zb. den stdout - Bei jedem Task Result wo abspeichern(ARA) - Inventory Plugins(Inventory File zb. von einer CMDB ziehen) - ...

Static Inventory File

- Typisch ein einzelnes ini file oder ein yaml file - wir werden dies hier so benutzten

Dynamic Inventory File (zb. Netbox)

- Als Inventory Plugin - Kann auch ein Program mit passenden STDOUT sein(JSON) - Wie im Datacenter ein Inventory Plugin das die Daten von Netbox bezieht
