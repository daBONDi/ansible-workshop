Ansible Facts(Cache)

  • Bei Ausführung fragt Ansible einmalig den Host ab(setup), diese Information können später für die Verarbeitung verwendet werden
    • Hostname
    • IP/Interface Einstellungen
    • OS Type/Version/..
    • Disklayout
    • Hardware
    • ...
  • Facts - Speicherung (Cache Plugins)
    • JSON/YAML File
    • Redis Server
    • Memcache DB
    • MongoDB
    • ...

https://docs.ansible.com/ansible/latest/plugins/cache.html#cache-plugins

  • Später dann mehr bei playbooks
  • mit addhoc mal ansehen
