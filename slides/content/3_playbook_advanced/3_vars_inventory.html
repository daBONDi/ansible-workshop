Inventory Group Vars / Host Vars

Variablen Files für Inventory Groups und Hosts sind immer relative zum Inventory Source zu definiert.

ansible-playbook site.yml -i inventory.ini

simple-project
├── inventory.ini
├── group_vars
│ ├── all
│ │ └── common.yml
│ ├── web
│ │ ├── webserver_config.yml
│ │ └── ufw.yml
│ └── database
│   ├── mysql.yml
│   └── ufw.yml
├── host_vars
│ ├── db1
│ │ └── db_settings.yml
│ ├── db2
│ │ └── db_settings.yml
│ ├── web1
│ │ └── web_settings.yml
│ └── web2
│   └── web_settings.yml
└── site.yml

group_vars / host_vars werden immer relativ vom inventory file erkannt!!

Inventory spezifisches Layout

ansible-playbook site.yml -inventory dev
ansible-playbook site.yml -i prod

project
