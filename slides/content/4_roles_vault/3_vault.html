# Ansible Vault - Symetrische Verschlüsselung (AES Chiper) von - Dateien - Text https://docs.ansible.com/ansible/latest/user_guide/vault.html#vault-payload-format-1-1-1-2 - Password wird bei Playbook Execution übergeben ```yaml # Fragt nach dem Passwort vor ausführung ansible-playbook myplaybook.yml --ask-vault-pass # Liest die Datei ein als Vault Password ansible-playbook myplaybook.yml --vault-password-file ".vaultpass" ``` - Kann über ENV Variable __DEFAULT_VAULT_PASSWORD_FILE__ übergeben werden > Aufpassen das Ihr KEIN Password File in das Sourcecontrol übergibt! > .gitignore verwenden!
# Ansible Vault - Password von Executable Python **vault_pass.py** ```python #!/usr/bin/python print 'my-ultra-secret-password' ``` Ansible erkennt ob vault-password-file als Executable markiert ist und führt dieses aus ```bash ansible-playbook --vault-password-file vault_pass.py ```
# Ansible Vault - Dateien ```yaml # Datei Verschlüsseln ansible-vault encrypt my_secret_file --vault-password-file mypass # Datei Entschlüsseln ansible-vault decrypt my_secret_file --vault-password-file mypass ``` Dateien werden automatisch von Ansible bei Verwendung in Modulen entschlüsselt!
