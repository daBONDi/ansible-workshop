# Windows - Connection Setup https://docs.ansible.com/ansible/latest/user_guide/windows_setup.html ## Setup Script ```powershell $url = "https://raw.githubusercontent.com/ansible/ansible/devel/examples/scripts/ConfigureRemotingForAnsible.ps1" $file = "$env:temp\ConfigureRemotingForAnsible.ps1" (New-Object -TypeName System.Net.WebClient).DownloadFile($url, $file) powershell.exe -ExecutionPolicy ByPass -File $file ``` - Selfsigned SSL Cert - Listener HTTPS auf 5986 - Basic Auth über HTTPS Enabled notes: - Setup Script von Ansible Repo - HTTPS Selfsign Certificate + SSL Listener - Basic Authentication - Link für WinRM Debugging
## WinRM Settings ```powershell # WinRM Listener Anzeigen # - Interface Listeners für den WinRM Service # - Interface # - Port winrm enum winrm/config/Listener # WinRM Service Settings # - Auth Methos # - Session Limits # - IP Filters winrm get winrm/config/Service winrm get winrm/config/Winrs ``` notes: - winrm config tool, wichtigsten befehle - zu kurze zeit um alles zu zeigen - Certificate sowie WinRM Settings über GPO verteilbar
